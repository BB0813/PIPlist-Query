功能特点

包管理

  • 检测已安装的 Python 包及其版本
  • 导出包信息为 JSON/YAML 格式
  • 生成包依赖关系图
  • 检查包版本与 requirements.txt 的匹配情况

输出文件：

  • Python库.xlsx - 已安装的 Python 包信息
  • dependency_graph.png - Python 包依赖关系图
  • 依赖匹配.xlsx - requirements.txt 的依赖匹配结果

虚拟环境管理

  • 创建新的虚拟环境
  • 查看现有虚拟环境列表
  • 激活指定的虚拟环境
  • 删除不需要的虚拟环境

注意：虚拟环境的创建和删除可能需要一定时间，请耐心等待

安全检查

  • 检查已安装包的安全警告
  • 导出安全检查报告
  • 实时显示检查进度

输出文件：

  • security_check_*.txt - 安全检查报告

性能监控

  • 实时监控 CPU 使用率
  • 内存使用情况跟踪
  • 磁盘使用状态监控
  • 导出监控数据为 CSV 格式

输出文件：

  • performance_monitor_*.csv - 性能监控数据
